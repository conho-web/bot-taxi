# bot-taxi — VK-бот для анонимных заказов такси

Node.js + [Bots Long Poll API](https://dev.vk.com/api/bots-long-poll/getting-started) ВКонтакте + SQLite.

## Возможности

- **Пассажир** — в ЛС: **«🚕 Заказать такси»** → маршрут. После ответа водителя — **«Да, едем»** / **Отмена**; в поездке переписка с водителем **в этом же диалоге с ботом**. Завершить поездку может **только водитель**.
- **Водитель** — в ЛС: **«🚗 Я водитель»** и позывной → заявка админу. После одобрения заказы в **беседе водителей** (кнопки **~ 3-5 / ~ 10 / ~ 20 мин**). Переписка и **«🏁 Завершить заказ»** — в личке с ботом.
- **Цены** — кнопка **«💰 Цены»**; текст в файле `prices.txt` (можно менять на сервере или через админку).
- **Админ** — в `.env` задаётся `ADMIN_VK_IDS`; в ЛС **«⚙️ Админ»** или `/admin`: заявки, блокировка/разблокировка водителей, смена цен.
- По нажатию кнопки в беседе пассажир получает предложение в ЛС с кнопками **Да, едем** / **Отмена**.
- После подтверждения водитель и пассажир пишут друг другу **только в личных сообщениях бота** (не в беседе водителей).
- Свободные сообщения в беседе водителей **пассажирам не пересылаются**.

## Настройка ВК

1. Создайте **сообщество**, включите **Сообщения сообщества**.
2. **Управление → Работа с API**: создайте ключ с доступом к сообщениям.
3. **Управление → Настройки → Работа с API → Long Poll API**:
   - Версия API: **5.199** (или совместимая).
   - Включите типы событий: **Входящее сообщение** (`message_new`), **Действие с сообщением** (`message_event`).
   - **Callback API можно отключить** — бот работает через long poll, публичный URL не нужен.
4. Добавьте бота-сообщество в **беседу водителей** (как участника с доступом к сообщениям). Узнайте `peer_id` беседы: напишите в беседу и посмотрите `peer_id` в логе `[message_new]` при запуске бота, затем укажите в `DRIVERS_PEER_ID`.

## Запуск

```bash
cp .env.example .env
# заполните .env
npm install
npm start
```

Переменные см. в `.env.example`.

### Цены

Отредактируйте `prices.txt` в корне проекта — бот читает файл при каждом нажатии **«💰 Цены»** (перезапуск не нужен).

### Админ

1. Узнайте свой VK `user_id` (из ссылки профиля или через API).
2. Добавьте в `.env`: `ADMIN_VK_IDS=ваш_id` (несколько — через запятую).
3. В ЛС с ботом: **«⚙️ Админ»** — заявки, блокировка одобренных водителей (список или VK id), разблокировка, смена `prices.txt`.

## Деплой на Timeweb (VPS)

1. Загрузите проект на сервер (git clone или SFTP).
2. Установите Node.js **18+** (`node -v`).
3. В каталоге проекта: `npm install`, заполните `.env`.
4. Запустите процесс так, чтобы он не падал при выходе из SSH:
   - **pm2** (рекомендуется): `npm i -g pm2 && pm2 start src/index.js --name bot-taxi && pm2 save && pm2 startup`
   - или **systemd** — unit с `ExecStart=/usr/bin/node /path/to/bot-taxi/src/index.js`
5. Публичный домен и HTTPS **не нужны** — бот сам опрашивает VK.
6. Опционально: `PORT=3000` — поднимется `GET /health` для проверки, что процесс жив (`curl localhost:3000/health`).

## Проверка

- В логах после старта: `[longPoll] сервер получен, ts=...`
- Напишите боту в ЛС — в логах появится `[VK update] { type: 'message_new' }`
- `GET /health` → `ok` (если `PORT` > 0)

### ЛС с сообществом и `peer_id`

В long poll для диалога с сообществом ВК может прислать `peer_id` как **`2000000000 + user_id`**, так и **`peer_id === user_id`**. Бот учитывает оба варианта и отвечает в **`messages.send`** тем же `peer_id`, с которого пришло сообщение.

### Интерфейс на кнопках

**Пассажир (ЛС):** Заказать / Цены / Я водитель / Помощь.  
**Водитель (одобрен):** Профиль / Цены / Помощь; в поездке — Завершить / Помощь.  
**Водитель (ожидает):** Профиль / Цены / Помощь — заказы в беседе недоступны до одобрения.  
**Водители (беседа):** на заказе — ~3-5 / ~10 / ~20 мин или **«✏️ Своё время»** (минуты или текст в ЛС). Можно брать заказы **по пути**, несколько параллельно.  
**Водитель (ЛС):** **«📋 Мои поездки»** — список, переключение (▶), **⏳** и **🏁** по каждому заказу; **«⏳ Ожидаю»** — один раз на заказ; переписка: `#номер текст`.  
Упрощённый режим: `SIMPLE_DRIVER_FLOW=true` — без «Завершить», старые поездки закрываются при «Ожидаю» у следующего клиента.  
«⏳ Ожидаю» — один раз на заказ, после нажатия кнопка исчезает.  
`MULTIPLE_PASSENGER_ORDERS` (по умолчанию вкл.) — пассажир может вызвать несколько машин подряд.

- `src/longPoll.js` — цикл опроса VK
